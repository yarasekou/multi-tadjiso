# Tadjiso

Gestion multi‑stations pour entreprises stations services.

## Stack
- Symfony
- PHP 8.2
- MySQL
- Bootstrap

## Fonctionnalités
- Gestion de plusieurs stations par entreprise
- Suivi des stocks de carburant
- Gestion des ventes et facturation PDF
- Tableaux de bord par station
